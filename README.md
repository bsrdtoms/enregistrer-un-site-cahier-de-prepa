# enregistrer-un-site-cahier-de-prepa
Ce projet ne vise pas à surcharger les serveurs. Il propose une copie complète d’un site Cahier de Prépa afin d’assurer un accès fiable aux ressources. Pensé pour les étudiants en prépa, souvent stressés et sans temps pour développer leur propre solution, il garantit une continuité d’accès en cas de problème.
